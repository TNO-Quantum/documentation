ml.classifiers.svm.kernels

Module containing kernel function used by the SupportVectorMachine.

tno.quantum.ml.classifiers.svm.kernels.get_kernel(kernel, gamma=None, degree=None, coef0=None)[source]

Given a kernel, which can be a string or callable, returns a kernel function.

Parameters:

  • kernel (str | Callable[ArrayLike, ArrayLike, float]) – Name of the kernel or a callable. If a string is given, choose from ‘rbf’, ‘sigmoid’, ‘poly’ or ‘linear’.

  • gamma (Optional[SupportsFloat]) – gamma parameter of the ‘rbf’, ‘sigmoid’ or ‘poly’ kernel.

  • degree (Optional[SupportsFloat]) – degree of the ‘poly’ kernel.

  • coef0 (Optional[SupportsFloat]) – Offset in the ‘sigmoid’ or ‘poly’ kernel.

Return type:

Callable[ArrayLike, ArrayLike, float]

Returns:

kernel function corresponding to the name, or the given callable.

Polynomial kernel for the SVM.

Parameters:

Return type:

float
