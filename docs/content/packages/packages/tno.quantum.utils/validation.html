tno.quantum.utils.validation.check_arraylike(x, name, *, ndim=None, shape=None)[source]

Check if the variable x with name name is an ArrayLike.

Optionally, check if the result has the specified number of dimensions.

Parameters:

  • x (Any) – Variable to check.

  • name (str) – Name of the variable. This name will be displayed in possible error messages.

  • ndim (Optional[int]) – Number of dimensions x should have. When ndim = 1, arrays with more than one dimension will be squeezed to remove dimensions of size one.

  • shape (Optional[tuple[int, ...]]) – Shape x should have.

Raises:

ValueError – When the provided input is not ArrayLike or does not have the correct shape or number of dimensions.

Return type:

ndarray[tuple[Any, ...], dtype[Any]]

Returns:

NDArray representation of x.

tno.quantum.utils.validation.check_path(x, name, *, must_exist=False, must_be_file=False, must_be_dir=False, required_suffix=None, safe=False)[source]

Check if the variable path with name name is a valid path.

Optionally, existence, file, and directory checks can also be performed.

Parameters:

  • x (Any) – Variable to check.

  • name (str) – Name of the variable. This name will be displayed in possible error messages.

  • required_suffix (Optional[str]) – If specified, the path must have this suffix.

  • must_exist (bool) – If True, the path must exist.

  • must_be_file (bool) – If True, the path must be a file.

  • must_be_dir (bool) – If True, the path must be a directory.

  • safe (bool) – If True and the path does not have the required suffix a ValueError is raised. Otherwise, if False, the suffix will be replaced to match the required_suffix.

Raises:

  • TypeError – If path is not a string or Path object.

  • ValueError – If path does not have the correct required_suffix.

  • OSError – If path does not exist while must_exist is True

  • FileNotFoundError – If path is not a file while must_be_file is True

  • NotADirectoryError – If path is not a directory while must_be_dir is True

Return type:

Path

Returns:

Path object representing path.
