utils.noise

This module contains various standard noise models.

These standard noise models can be used by themselves or through the NoiseConfig class.

tno.quantum.utils.noise.depolarizing(device, **kwargs)[source]

Depolarizing noise model.

This noise model applies DepolarizingChannel after certain gates. If the argument p is a float, a depolarizing channel with probability p is applied to each wire after every gate. If the argument p is a dict[str, float], then a depolarizing channel is applied to each wire after every gate whose name is a key of p, with probability the corresponding value.

Parameters:

  • device (Device) – Device to transform.

  • p – Probability of the DepolarizingChannel. Default is 0.01.

  • kwargs (Any) – Not used.

Return type:

Device
